# Project 3 — VPC Setup

## What I Built
A custom VPC with public and private subnets,
internet gateway and route tables on AWS.

## Architecture
Internet → Internet Gateway → VPC
├── Public Subnet (10.0.1.0/24) → Web Servers
└── Private Subnet (10.0.2.0/24) → Databases

## AWS Services Used
- VPC
- Public & Private Subnets
- Internet Gateway
- Route Tables

## What I Learned
- How VPC isolates your network on AWS
- Difference between public and private subnets
- How internet gateway connects VPC to internet
- How route tables direct traffic
