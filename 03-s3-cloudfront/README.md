A fully functional car website hosted on AWS S3 with 
static website hosting enabled and public access 
configured using bucket policy.

## Architecture
Internet → S3 Bucket (Static Website Hosting) → Car Website

## AWS Services Used
- Amazon S3
- Bucket Policy (Public Access)
- Static Website Hosting

## What I Did

- Uploaded HTML, CSS, JS and image files
- Enabled static website hosting
- Configured bucket policy for public access
- Troubleshot 403 and 404 errors
